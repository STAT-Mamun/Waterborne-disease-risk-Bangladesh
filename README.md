# Predicting Household Risks of Waterborne Diseases in Flood-Affected Areas Using Machine Learning

This repository contains the data and code used in the study  
**“Predicting Household Risks of Waterborne Diseases in Flood-Affected Areas Using Machine Learning:
A Study of Noakhali and Feni, Bangladesh.”**

The study applies machine learning techniques to household survey data to identify
key risk factors associated with waterborne diseases in flood-prone regions.

---

## Overview

A total of 501 household survey responses were collected from flood-affected areas
of Noakhali and Feni, Bangladesh. After data preprocessing and cleaning, all 501
records were retained for final analysis. The dataset includes environmental,
socio-economic, infrastructural, behavioral, and health outcome variables.

---

## Data Description

The dataset consists of the following variable groups:

- **Environmental factors** 
- **Socio-economic characteristics** 
- **Infrastructural and health-related variables**
- **Behavioral and preventive practices**
- **Health outcome**: suffered from waterborne disease (binary)

All data have been anonymized to protect respondent privacy.

A detailed description of variables and coding schemes is provided in
`data/data_dictionary.xlsx`.
